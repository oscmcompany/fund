## About

The **fund** repository holds the resources for the Open Source Capital Management platform.

The project is actively a work-in-progress.  

## Project

Below are resources for the project and repository.

### Setup

Run the commands below in the root directory.  

```sh
# Install devenv — see https://devenv.sh/getting-started/
devenv shell      # enter development environment
devenv up         # start local services
devenv tasks list # list available tasks
```

### Principles

An unordered and non-exhaustive list we work towards:  

> Test in production  
> Always roll forward  
> Systems over process  
> No code is good code  
> Never write documentation  
> Be explicit  
> Git is truth
